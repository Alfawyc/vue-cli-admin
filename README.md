# vue-cli-admin

# 项目运行

- 安装vue-cli
- Node 版本要求
    vue-cli 需要Node.js 8.9或更高版本(推荐8.11.0+)

```bash
# 安装vue-cli
npm install -g @vue/cli
```

- 切换到更目录下,安装依赖
```bash
npm install
```

- 开始开发页面

```bash
npm run serve
```

- 发布
```bash
npm run build
```

# 其他说明

-   模板基于 vue-cli3 创建, 全局安装 vue-cli3 后 bash 运行

```bash
vue ui
```

可以打开图形化界面管理项目
